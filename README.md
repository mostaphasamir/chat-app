# Chat Application

## Overview
This app is built with Flutter & firebase service .
* GetX library for state management
* MVC (Model-View-Controller) architectural pattern.
* Firebase Authentication
* Firebase  Cloud Firestore
* Firebase  Storage
* Image Packer
